Algemeen

GoHugo is een leuke manier om een site te creëren maar moeilijk om aanpassingen te maken naar keuze. De thema’s waarmee je kan werken zijn beperkt. Om aanpassingen te maken moet je al kennis hebben van css, bootstrap,… Bij Wordpress kan de gebruiker op een relatief eenvoudige wijze de inhoud (content) wijzigen en/of aanvullen zonder dat daar technische kennis voor vereist is.

Info

Om GoHugo te kunnen draaien is er geen host nodig. Bij Wordpress wel maar daar zit je al met een geïntegreede localhost.

Mijn persoonlijke ervaring met GoHugo

Ik ben niet zo goed in codetaal dus draagt GoHugo niet mijn voorkeur. Met Wordpress heb ik nog niet veel gewerkt maar het ziet er gemakkelijker uit dan GoHugo. Op zicht is Hugo wel leuk om in te werken maar het wordt ingewikkeld bij geavanceerde aanpassingen.

Bijgeleerd

Bij het maken van de GoHugo site heb ik wel wat bijgeleerd zoals hoe ik in bepaalde thema’s een menu kan maken om naar deelpagina’s te gaan. Structuur? Ik ben gestart met in mijn content een menu te creëren voor stage en GIP. Vervolgens ben ik in layout verder gegaan met de partials wat aan te passen. Dan ben ik in mijn GIP map gegaan om daar dit verslag in te verwerken.
