Review of a network tool

Case study: Engels

VPN: good or bad? Safe or Unsafe?

    I’m writing my review about VPN (Virtual Private Networking). It’s a way of connecting two computers over a wide network. VPN is a cheaper way of connecting 2 computers while WAN (Wide Area Network) is more expensive.
    The process of this VPN goes like this: first a computer is connected to the internet so that it can broadcast a signal through the internet to a program that is located on a host (where the host is a PC or server or online software). Next the software creates (by using an OS or a virtual sheet over the OS that is already on the PC) a connection back to the original PC so the first PC can take over the other PC. Mostly there is also a firewall between the server and the PC so that hackers can’t get in that easy. Sometimes (like I said) people use virtual sheets and they do that because they want to use as few mb/s as possible so that it becomes even cheaper. The thing is that when there is a virtual sheet there doesn’t need to be a refresh rate of 60 frames per second but more like 10fps. Thanks to the recent growth of internet not only will VPN be more common but it will only lower the price of such one VPN. Like I said before it’s not super cheap because of the firewall expenses you’ll have to pay but once it’s up it will only make your wallet feel better. Main reason of firewalls is that in VPNs people connect 2 computers with each other which attracts hackers because with the right software people can take over a PC. The next thing about VPN is something my company I worked at had. It was a MaTMa. MaTMa technology links your machines together via a wireless connection between an industrial modem and a server. This communication is done through a special MaTMa SIM card.

Pros

  • I am inpressed by the fact that it is so cheap.
  • A big plus is that you can use it to work from home.
  • But the best thing is that you can keep an eye on your coworkers (as a boss) to see if they do their job right.

Cons

  • a disadvantage is that you have to spend that extra money to insert firewalls and such security measures.

Conclusion

    VPN is a must if people actually need it. Don’t use it if you don’t know anything about it. It can be dangerous however, it can also be very useful for some people.
